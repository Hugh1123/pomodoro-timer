# 🍅 Pomodoro Timer - 番茄鐘計時器

一個功能豐富、美觀易用的番茄工作法計時器，幫助你提升專注力和效率。

## ✨ 功能特色

### ⏰ 計時功能
- **三種模式**：工作、短休息、長休息
- **自動循環**：完成設定輪數後自動進入長休息
- **視覺反饋**：
  - 圓形進度環
  - 倒數計時顯示
  - 脈動動畫效果
  - 顏色主題切換（工作=紅、短休息=綠、長休息=藍）

### 🔔 提醒通知
- **音效提醒**：時間到時播放和弦音效
- **桌面通知**：支援瀏覽器原生通知
- **頁面標題**：即時顯示剩餘時間

### 📝 任務管理
- 新增待辦事項
- 勾選完成狀態
- 刪除任務
- LocalStorage 持久化儲存

### 📊 統計追蹤
- 總完成番茄數
- 今日完成數
- 累積工作分鐘數
- 自動保存到本地

### ⚙️ 自訂設定
- 工作時間（預設 25 分鐘）
- 短休息時間（預設 5 分鐘）
- 長休息時間（預設 15 分鐘）
- 番茄鐘輪數（預設 4 輪）

### 🎨 UI/UX 設計
- 漸層背景（隨模式切換）
- 毛玻璃效果（backdrop-blur）
- 平滑動畫過渡
- 響應式設計
- 勵志語錄顯示

## 📖 使用說明

### 1. 開始計時
1. 選擇模式（工作/短休息/長休息）
2. 點擊「▶️ 開始」
3. 專注工作直到時間結束

### 2. 管理任務
1. 在右側任務清單輸入待辦事項
2. 按 Enter 或點擊 ➕ 新增
3. 完成後勾選方框
4. 不需要的任務可點擊 🗑️ 刪除

### 3. 調整設定
1. 點擊「⚙️ 設定」
2. 調整各模式的時間長度
3. 設定番茄鐘輪數
4. 點擊「保存」

### 4. 查看統計
- 畫面下方顯示累積統計
- 包含總完成數、今日完成數、工作時間

## 🎯 番茄工作法說明

番茄工作法是一種時間管理技巧：

1. **工作 25 分鐘**（一個番茄鐘）
2. **休息 5 分鐘**（短休息）
3. 重複 4 次後
4. **休息 15-30 分鐘**（長休息）

這種方法可以：
- ✅ 提升專注力
- ✅ 減少疲勞
- ✅ 提高生產力
- ✅ 更好的時間意識

## 🎨 設計亮點

- **動態背景**：工作時紅色、休息時綠色/藍色
- **圓形進度環**：SVG 繪製的流暢進度條
- **脈動效果**：計時中的呼吸動畫
- **毛玻璃卡片**：現代化 UI 設計
- **懸停縮放**：按鈕互動反饋
- **自訂動畫**：pulse、bounce、rotate

## 🔔 瀏覽器權限

首次使用時會請求：
- **通知權限**：用於桌面提醒（可選）

## 💾 數據儲存

所有數據儲存在瀏覽器 LocalStorage：
- 任務清單
- 統計數據
- 設定偏好

清除瀏覽器數據會重置所有記錄。

## 🌟 特色功能

1. **Web Audio API 音效**：無需外部音檔，使用原生 API 生成和弦
2. **即時語錄**：每次完成番茄鐘顯示不同勵志語
3. **智能通知**：完成時自動切換到下一模式
4. **輪數追蹤**：清楚顯示當前進度
5. **一鍵重置**：隨時重新開始
